# Mock UI Variants

Static visual mockups for comparing product directions before changing the real Electron UI.

Variants:

- `notion.html`: quiet, soft, minimal, Notion-inspired productivity workspace
- `crypto.html`: dark, sharp, high-contrast, neon-accent media terminal
- `lightroom.html`: conservative, dense, practical pro-media browser

Open `index.html` in a browser to compare all three.
